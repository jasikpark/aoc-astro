# Today's day one

We need to find the elf with the most calories from the [input](/aoc/inputs/01.txt)

## Part I

The inputs are formatted as numbers delimited by single lines, so we can import the file with vite's raw import support.

import input from "/aoc/inputs/01.txt?raw";

```ts
import input from "/aoc/inputs/01.txt?raw";
```

<pre>
  <code>{input}</code>
</pre>

we can then `.split(\n\n)` and `.sum()` and `.sort()`.

```ts
const solveList = input
  .split("\n\n")
  .filter((x) => !!x)
  .map((elfPack, elfNumber) => {
    return {
      elfPackCalories: elfPack.split("\n").reduce((sum, item) => {
        return sum + parseInt(item);
      }, 0),
      elfNumber,
    };
  })
  // get the largest number
  .sort((a, b) => b.elfPackCalories - a.elfPackCalories);
const solvePartI = solveList[0];
```

## Part II

Part II wants us to figure out how much the top three Elves are carrying.

Luckily we can just sum the first three elves in our lists's calories carried to find this out.

```ts
export const solvePartII = solveList
  .slice(0, 3)
  .reduce((sum, { elfPackCalories }) => sum + elfPackCalories, 0);
```
